Orange Box

DiGiCo I/O in a 2U rack unit which uses DMI

Orange Box With DiGiCo’s compact 2U Orange Box, you can use DiGiCo Multichannel Interface (DMI) cards to create audio paths over whatever interface you desire. The Orange Box has 2 PSUs for redundancy, and 2 slots to accommodate any of our DMI cards that allow you to convert pretty much any format to another.

DMI-OPTO

DMI Interface Options

This card allows Optocore audio over HMA, OpticalCon or ST connectors.

DMI-DANTE

DMI Interface Options

This card provides 64 input and 64 output channels at 48KHz and 32 input and 32 output channels at 96KHz. It is provided with Primary and Secondary (backup) Gigabit Ethercon ports for connection to the Dante network.ndancy, and 2 slots to accommodate any of our DMI cards that allow you to convert pretty much any format to another.

DMI-DANTE64@96

DMI Interface Options

The DiGiCo DMI-DANTE64@96 allows 64 channels in and 64 channels out at both 48kHz and 96kHz. It has a Primary and a Secondary port, both of which are Ethercon ports, and an RJ45 Control port. The card has two modes that can be changed in Dante Controller. In Switched Mode, the three ports are switched together.

DMI-KLANG

DMI Interface Options

This card provides ultra low latency KLANG immersive processing for up to 16 2-channel mixes with 64 inputs at 48kHz or 96kHz.

DMI-MADI-B

DMI Interface Options

This card can be used to connect a Standard MADI stream (64 channels in and out) at 48KHz or 96KHz or an SD Series DiGiCo Rack with the appropriate connector (D2-Rack, SD-Rack, SD-MINIRack)

DMI-MADI-C

DMI Interface Options

This card can be used to connect another DiGiCo device that uses RJ45 Standard MADI (64 channels in and out) at 48KHz or 96KHz or an SD Series DiGiCo Rack with the appropriate connector (D-Rack, D2-Rack)

DMI-WAVES

DMI Interface Options

This card will provide 64 input and 64 output channels at 48kHz or 96kHz to the SoundGrid Network with 2 Ethercon ports.

DMI-AMM

DMI Interface Options

This card offers both the SD-Range and the S-Series consoles up to 64 channels of automatic microphone mixing which can be used in any console input channel.

A-Net Card

Aviom D-16c A-Net Card

Aviom are a popular manufacturer of personal monitoring systems. DiGiCo offer two Aviom rack cards (one compatible with SD racks and the other compatible with D2 racks) that make it easy to incorporate Aviom’s personal mixers into a DiGiCo system. Both cards provide 16 output channels (8 stereo) at 48kHz and support Aviom’s proprietary A-Net Pro16 protocol. Both feature 1 Ethercon port and DIP switches to control stereo output selection.
